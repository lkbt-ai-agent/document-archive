# 생성 문서 이력과 계보 상세 TODO

## 범위

AI 생성 문서의 처리 이력과 계보 표시를 더 자세히 제공합니다.

## 백엔드

- 생성 문서는 검색 가능하게 인덱싱되고 계보도 저장합니다.
- 단계별 처리 로그나 상태 전이 이력은 남기지 않습니다.
- 필요 시 작업 단계, 재시도, context 축소 이력, 제목 생성 fallback 여부를 기록해야 합니다.

## 프론트엔드

- 생성 문서의 작업, 모델, 출처 문서, 프롬프트는 볼 수 있습니다.
- 제공자, 생성 파라미터, 워크플로 DNA, 출처 청크 ID/내용 상세는 표시하지 않습니다.
- 출처 청크를 눌러 원본 문서/청크로 이동하는 UI가 필요합니다.
